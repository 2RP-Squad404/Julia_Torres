# BigQuery
Ele é projetado para permitir a execução de consultas SQL rápidas e eficazes em grandes volumes de dados fornecido pelo Google Cloud.

### Características:
* Totalmente gerenciado;
* Escalabilidade;
* Alto Desempenho;
* suporte a SQL;
* Integração;
* Modelo de Preços Baseado em Uso.

### Benefícios:
* Velocidade;
* Facilidade de Uso;
* Custo-Efetividade;
* Flexibilidade.
